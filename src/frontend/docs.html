Language Tandem App Documentation

Welcome to the Language Tandem App documentation. This guide will help you understand how to use the application effectively for language learning with a partner.

Note: This is a prototype version of the application. Some features may be limited or use mock data when the backend is not available.

Getting Started

Creating an Account

To use the Language Tandem App, you need to create an account or log in:

  1. Click the "Login" button in the top-right corner of the application.
  2. Enter your email address when prompted.
  3. If you're a new user, you'll be asked to provide your name.
  4. After logging in, you'll be directed to complete your profile.

Completing Your Profile

A complete profile helps you find suitable language partners:

  1. Navigate to the "Profile" page.
  2. Fill in your native language and the language you're learning.
  3. Select your proficiency level in the language you're learning.
  4. Add your interests and learning goals to help find compatible partners.
  5. Click "Save Profile" to update your information.

Features

User Management

The User Management system allows you to:

  • Create and manage your user profile
  • Specify your native language and the language you're learning
  • Set your proficiency level
  • Add interests and learning goals
  • Find potential language partners based on complementary language skills

Exercise System

The Exercise System provides various language learning activities:

  • Flashcards: Practice vocabulary with digital flashcards
  • Quizzes: Test your knowledge with multiple-choice questions
  • Conversations: Practice dialogues with guided conversation prompts
  • Pronunciation: Improve your pronunciation with speaking exercises

To use exercises:

  1. Navigate to the "Exercises" page.
  2. Select an exercise type from the available options.
  3. Follow the on-screen instructions to complete the exercise.
  4. Submit your answers to receive feedback.

Progress Tracking

The Progress Tracking feature helps you monitor your language learning journey:

  • View statistics on completed exercises
  • Track your daily streak of language learning
  • See your progress in different skill areas (vocabulary, grammar, etc.)
  • Set and monitor learning goals

Cultural Learning

The Cultural Learning section enhances your understanding of the language's cultural context:

  • Cultural Notes: Learn about traditions, customs, and cultural practices
  • Idioms: Discover common expressions and their meanings
  • Fun Facts: Explore interesting facts about the language and its speakers

Offline Mode

The Language Tandem App includes limited offline functionality:

  • User data is cached in localStorage for persistence between sessions
  • When the backend is unavailable, the app generates mock data for testing
  • Changes made in offline mode will be synchronized when the connection is restored (in future versions)
Warning: Offline mode is primarily for testing purposes. For the best experience, use the application with an active internet connection and backend server.

Troubleshooting

Common Issues

Issue Solution
Cannot log in Check your internet connection. If the backend is unavailable, the app will create a mock user for testing.
Exercises not loading Verify that you've completed your profile. Some exercises require language preferences to be set.
Progress not saving Ensure you're logged in and have an active internet connection. In offline mode, progress is stored locally.
UI display issues Try refreshing the page or clearing your browser cache. The app works best in modern browsers like Chrome, Firefox, or Edge.

Clearing Local Storage

If you encounter persistent issues, you may want to clear the application's local storage:

  1. Open your browser's developer tools (F12 or right-click and select "Inspect")
  2. Navigate to the "Application" or "Storage" tab
  3. Select "Local Storage" and find the entry for this application
  4. Click "Clear" or delete specific items
  5. Refresh the page
